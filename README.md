## ✨ About

**UwU Invaders** is a fast-paced, browser-based shooter inspired by retro classics like Space Invaders, but infused with pure **UwU chaos energy**.

Dodge. Shoot. Survive.  
Or get absolutely overwhelmed by the cutest invasion imaginable.
